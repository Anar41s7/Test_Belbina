# Тест Белбина — диагностика командных ролей

Онлайн-опросник для определения командных ролей по методике Белбина.

## 📋 О проекте

Тест состоит из 7 разделов, в каждом из которых нужно распределить 10 баллов между 8 утверждениями.
В конце теста формируется профиль командных ролей пользователя.

## 🚀 Функционал

- Пошаговое прохождение теста
- Валидация суммы баллов (должно быть ровно 10)
- Прогресс-бар
- Автоматический подсчёт баллов по 8 ролям (Исполнитель, Координатор, Формирователь, Мыслитель, Разведчик, Оценщик, Коллективист, Доводчик)
- Определение ведущей роли
- Полное описание каждой роли
- Экспорт результатов в PDF
- Отправка результатов на почту преподавателя
- Адаптивный дизайн

## 🛠️ Технологии

- React 19
- Vite
- CSS Modules
- jsPDF + html2canvas (генерация PDF)
